# Contractor Lead Conversion System — Outreach Target List Format V1

## Objective
Define the simple list format for prospecting targets.

The list should help Jason's wife quickly work through leads without confusion.

---

# Recommended Columns
- Company Name
- Trade
- City
- State
- Website
- Main Phone
- Owner / Manager Name
- Owner / Manager Email
- Google Ads / LSA Likely? (Yes / No / Unsure)
- Notes
- First Call Attempt Date
- Call Outcome
- Follow-Up Needed
- Email Sent Date
- Demo Booked? (Yes / No)
- Status

---

# Suggested Status Options
- New
- Called
- Left Voicemail
- Spoke to Gatekeeper
- Spoke to Owner
- Email Sent
- Follow-Up Needed
- Demo Booked
- Not Interested
- Closed Lost

---

# Best Notes to Capture
- do they run ads?
- do they sound busy enough to miss calls?
- who handles leads?
- what objection came up?
- best callback time?

---

# Best Practice
Keep the list simple.
The point is to:
- know who was contacted
- know what happened
- know what next step is needed
