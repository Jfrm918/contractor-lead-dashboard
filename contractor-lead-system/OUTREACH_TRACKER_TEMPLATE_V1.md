# Contractor Lead Conversion System — Outreach Tracker Template V1

## Objective
Simple tracker structure for daily outreach and follow-up.

## Columns
- Date Added
- Company Name
- Trade
- City
- Website
- Main Phone
- Owner/Manager Name
- Email
- Lead Source Guess (Google Ads / LSA / Organic / Unknown)
- First Call Attempt
- First Call Outcome
- Email Sent
- Follow-Up Call Date
- Follow-Up Outcome
- Demo Booked
- Demo Date
- Proposal Sent
- Pilot Status
- Notes

## Pilot Status Options
- New
- Working
- Warm
- Demo Booked
- Proposal Sent
- Pilot Pending
- Closed Won
- Closed Lost
