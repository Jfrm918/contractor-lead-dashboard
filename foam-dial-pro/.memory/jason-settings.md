# Jason's Baseline Settings — Spring (April-May, Tulsa OK)

## Rig Settings
- Hose Temp A: 145F
- Hose Temp B: 145F
- Drum Temp A (A-side): 95F
- Drum Temp B (B-side): 100F
- Note: "All the way across at 145" = both hose heaters matched

## Foam System
- Enverge EasySeal .5 (open cell, 0.5 lb density)
- R-3.8 per inch
- Target yield: ~20,000 BF per set
- 1 set = 48 gallons
- 1:1 mix ratio by volume

## Typical Production
- 1 to 1.5 sets per day
- Sometimes as low as 20 gallons, sometimes 60
- Sprays: metal buildings, underfloors, stemwalls, wood substrate, new construction, retrofit
